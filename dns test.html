اسکنر DNS برای کانفیگ VLESS → WireGuard

این صفحه بدون پایتون، بدون سرور، بدون افزونه و فقط با مرورگر اجرا می‌شود. تست از داخل همان مرورگری انجام می‌شود که با VPN باز شده است.

مرورگر اجازه ICMP ping یا UDP DNS خام نمی‌دهد. عددها «Latency مرورگر به endpoint همان DNS» هستند، نه ping خام سیستم. برای انتخاب DNS بهتر داخل همین مسیر VPN کافی است.
